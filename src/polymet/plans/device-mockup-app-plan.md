# Device Mockup Generator App

## User Request
Create a web app that allows users to:
1. Select from a range of device mockups (phone, tablet, laptop, etc.)
2. Upload image files (png, jpg, gif) or video files
3. Crop the uploaded file
4. Process the file with a loading indicator
5. Download the resulting gif or mov file
